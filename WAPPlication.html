Projet 1

WApplication

WApplication, est une plateforme de Recrutement

Développement frontend, backend et déploiement d'une plateforme de recrutement .

Stack utilisées :

  • Frontend : React , Redux , materiel ui , html , css , webRTC.
  • backend : node , express .
  • base de données : mongoDB .

La plateforme permet de :

>

Pour le candidat :

  • -Créer son profil.
  • -Envoyer des pièces jointes.
  • - Consulter les offres.
  • -Enregistrer des vidéos pour se présenter.

Pour le recruteur :

  • -Créer son profil.
  • -Créer des jobs Offers.
  • -Télécharger des pièces jointes
  • - Visualiser la vidéo du candidat.
